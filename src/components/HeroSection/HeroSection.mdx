# Hero Section

Introduces a page or section with a clear heading and supporting subheading.

## Guidelines

- Use a short, specific heading and keep the subheading concise so the message is easy to scan.
- Enable `hasBackground` only when this section needs stronger visual emphasis than surrounding content.
